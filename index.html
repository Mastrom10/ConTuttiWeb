Servicios

Modalidad de Servicio

La modalidad del servicio será bandejeo. Nuestra moza irá ofreciendo a todos los invitados las diferentes preparaciones, así como también dejará bandejas en las distintas mesas para que los invitados puedan servirse. Nosotros llevamos todos los elementos necesarios para el servicio, incluyendo horno pizzero, mesas de trabajo, tablas de pernil, servilletas y servilleteros para las mesas, tablas, bandejas, etc.

Duración del Servicio

El servicio tendrá una duración de entre 3 a 4 horas. Nuestro personal llegará al evento 30 minutos antes que los invitados para realizar los preparativos previos.

Modalidad de Pago

Para reservar nuestros servicios, requerimos un 50% del pago total. El restante 50% se deberá abonar antes de comenzar el evento.

Personal

Nosotros llevamos Maestro Pizzero, cocinero Fileteador, y Mosos/as necesario según la cantidad de invitados.

Servicio de Bebidas

La modalidad del servicio será bandejeo. Nuestras mozas irán ofreciendo a todos los invitados las diferentes preparaciones, así como también dejarán bandejas en las distintas mesas para que los invitados puedan servirse. Nosotros llevamos todos los elementos necesarios para el servicio, incluyendo horno pizzero, mesas de trabajo, tablas de pernil, servilletas y servilleteros para las mesas, paneras, tablas, bandejas, etc. En el caso de las bebidas, nuestros mozos pueden encargarse de llevar jarras con bebida a las diferentes mesas y Hielo. (bebidas y hielo no incluido)
